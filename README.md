# Multi-Agent AI Task Manager

This project is a multi-agent AI system that helps users manage tasks, notes, and basic workflows.

## Features
- Main Agent to handle user input
- Task Agent for managing tasks
- Notes Agent for storing notes
- API-based system using Node.js
- Simple in-memory database

## How it Works
The main agent receives user input and routes it to appropriate sub-agents like task agent or notes agent.

## Technologies Used
- Node.js
- Express

## Outcome
This project demonstrates coordination between multiple agents and basic workflow handling.
